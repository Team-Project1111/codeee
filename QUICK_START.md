# 🚀 Kala Heritage - Quick Start Guide

## ⚡ Instant Setup (30 seconds)

```bash
# 1. Install dependencies
npm install

# 2. Start the platform (auto-setup)
./start.sh
```

**That's it!** 🎉 Open http://localhost:3000 in your browser.

## 🎯 What to Try

### 1. Explore the Platform
- Browse different artforms (Warli, Madhubani, Pithora, etc.)
- Click on artist profiles to see their work
- Use the search feature to find specific art
- Filter artworks by different categories

### 2. Create an Account
- Register as an Art Enthusiast or Artist
- Complete your profile with interests
- Start following artists and liking artworks

### 3. Interactive Features
- Like artworks and see real-time updates
- Comment on artworks to engage with artists
- Follow artists to build your network
- Search for specific artists or artworks

### 4. Artist Features (Login as Artist)
- View your artist dashboard
- See your artwork portfolio
- Check engagement analytics
- Manage your profile and bio
